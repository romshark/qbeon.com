Digital data has become the foundation of our modern life and society. People, organisations and machines produce a lot of data that must be stored in a safe and reliable place. Our data must be available for us wherever we go on any device we use.

Computer technology has evolved and humanity entered the new era of cloud computing. Servers all over the world interconnected in a gigantic, distributed network form digital clouds.

Cloud computing addresses the Big Data problem. When the amount of data to be processed or stored exceeds the limits of a single machine then there is a need for technology that distributes workloads and data sets among dynamic clusters of machines on the network.

The Cloud

The Cloud represents applications located on the internet that abstract away complex computational infrastructures and provide only a simple user interface.

Applications like Dropbox, Google Drive, Microsoft OneDrive and many others represent the SaaS (Software as a Service) category of The Cloud, they abstract away the complexity of distributed, fault tolerant and universally accessible file storages.

There is a major problem

We lose control of our data we upload onto the cloud. It becomes accessible to strangers and may easily be viewed, analyzed, altered, removed and used at their own discretion without our permission.

The cloud doesn't belong to us and therefore we cannot control it. Signing up for an account we are signing an agreement on the terms of use of proprietary services policies of which may change at any time.

The cloud is just
someone else's computer

To preserve our privacy and protect our personal data we build our own file storages for home and corporate usage. While big companies, organisations and government agencies run their own IT infrastructures maintained by professional IT specialists smaller companies and average users simply can't afford running their own IT departaments.

Existing personal storage devices
aren't solving this problem yet

Existing personal storage devices aren't solving this problem yet

Network attached personal storage devices have gained popularity during the rise of the cloud, they are however still very difficult to setup and use though, they require advanced technical skills, they still aren't scalable and they're still not secure enough.

Their graphical user interfaces are not consistent, look different and not always offer the same set of functions across all the different platforms. The filesystems of existing storage devices lack the necessary features for extensive collaboration and control.

It is therefore not surprising that we still have to choose between rich user experience and our own security.

We want to solve this problem

We want to solve
this problem

Cloud technology is a powerful tool, yet it's power isn't accessible to everyone. Building private cloud applications for data processing and storage is a complex and expensive task.

There is a need for a flexible cloud computing platform that is powerful yet controllable, dynamically scalable and secure. That's why for more than a year we work on a technology that eases the development of private cloud applications.

Besides the Big Data problem there is a growing demand for a simple and secure personal file storage solution. That's why we designed a network attached personal file storage device based on our technology, that meets the latest requirements.

Introducing the Qube

A compact network attached file storage device
for home and small businesses

A compact network attached
file storage device for home

It is very easy to setup and works right out of the box. Simply connect it to your router, install the Qube app on your smartphone, tablet or computer and your Qube is ready for use!

Sharing of files and folders with your friends, your family and your colleagues over the Qube is private, simple and intuitive. High level of security is ensured through advanced encryption of disk and network communication.

A modern graphical user interface provides a consistent user experience and the same functionality across all major platforms like Windows, Mac, Android, IOS and Linux.

We developed a seamless remote access technology that automatically connects to the Qube from anywhere over the internet without any network configuration right out of the box. Not only does our technology provide intuitive remote access capabilities, it also protects the Qube from different kinds of external threats.

Powered by QubeOS

The universal platform for cloud applications

The universal platform
for cloud applications

QubeOS is an open source platform for distributed, fault tollerant and secure private cloud applications that provides a layer of abstraction for distributed systems programming, facilitating the development of scalable applications.

Applications running on top of QubeOS are isolated, given a well defined set of system permissions. They can provide interactive, cross platform graphical user interfaces and also be packaged and published on the official app store.

Applications build on top of the QubeOS API are dynamically scalable. You can run these on small standalone machines, clusters of machines and even globally distributed groups of machines.
